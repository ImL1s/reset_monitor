# Board page overrides

Inherits `design-system/MASTER.md`.

## Layout
- Hero bento header with live counts (confirmed / pending / monitored)
- Live monitors grid: 1 / 2 / 3 columns by breakpoint
- Coverage section for `not_monitored` providers (compact cards)
- Status legend + disclaimer footer

## Card
- Icon + label + status pill (never color alone)
- Hover border accent on pointer devices only
- Touch targets ≥ 44px (icon container, copy button, refresh)

## Loading
- Skeleton placeholders matching grid columns (no blank flash)
